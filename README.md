# AudiBit #
## Music Sharing Service ##

## What is it going to do? ##
Audibit is a music sharing service in which aspiring and upcoming artists can upload their musical creations to share with the rest of the world. Musicians and non-musicians alike can use the service. Musicians will select a 15-second clip of their entire song to be featured in the “feed” for the users of the service. A user can “like” the clip that they’re listening to. If they want to hear the entire song, they will we able to go to the artist’s profile in which they can listen to the whole song.

## Who is the target Audience? ##
Musicians and people interested in finding new hip music to listen to. This is intended to be where the bleeding edge of music will be posted, not many “big” artists will use it because they’re already using Pandora, Spotify, Amazon Music, ITunes, etc.

## What kind of data will it manage? ##
The application will handle user profiles as well as audio files, images, and any other relevant data  needed for music sharing services.
